From uploading files, sharing, using metadata or extracting custom usage reports.
Before you get started make sure you have API access enabled and your account ID (used in some of the calls). For more inquiries contact developers@lookat.io

Projects

Description

Projects are the highest level entity in LookAt. Projects can be only created on the dashboard and cannot be nested under other projects or folders.
Any action in LookAt must start by creating a Project for the simple reason that content can only be uploaded into projects rather than directly to the Dashboard.
The Project object inherits from a Folder object and this is why it contains both ProjectID and FolderID properties. Keep this in mind since some of the queries require both or one of those ID's.

Post Create New Project

Use this call to create a new project on-behalf of a user

Post https://app.lookat.io/external_api/v1/payinggroups/{AccountID}/projects/post
Data Params:
{ Folder: {FolderName: "my project"}, ParentID: {FolderID} }
Header:
{ 'Authorization': 'Bearer {access_token}', userEmail: 'janedoe@lookat.io', 'Content-Type': 'application/json'}

Upload Files and Versions

Description

LookAt utilizes a CDN technology for both streaming and uploads. This is why uploading files or versions requires 2 API calls:
1. Retrieving a designated URL for the upload call.
2. Upload to the designated URL.

Sharing Files

Description

Each file in LookAt can be shared in unlimited number of private discussion groups.
Sharing a specific file in LookAt actually means you add a user to a certain group. Every file contains at least one group (Created automatically when a file is uploaded).

get Get List of Groups for a File

Use this call to retrieve the groups list for a file

Get https://app.lookat.io/external_api/v1/projects/{projectId}/videos/{fileId}/getVideoGroups
Header:
{ 'Authorization': 'Bearer {access_token}', userEmail: 'janedoe@lookat.io'} // userEmail - file owner
Sample Code (Javascript):

post Create New Group

Use this call to create new private discussion group

Post https://app.lookat.io/external_api/v1/projects/{projectId}/videos/{fileId}/group
Data Params: (Object)
{ Title: {group_name} }
Header:
{ 'Authorization': 'Bearer {access_token}', userEmail: 'janedoe@lookat.io'} // userEmail - file owner
